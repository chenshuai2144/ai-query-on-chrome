# 🐛 [BUG] hideInForm配置无效

### 🐛 bug 描述

hideInForm配置无效，筛选表单依然出现该列

### 📷 复现步骤

1. 配置hideInForm为true
2. 在筛选表单中仍然能看到该列

### 🏞 期望结果

hideInForm配置有效，筛选表单不显示该列

### 💻 复现代码

```javascript
import { Table } from "antd";

const columns = [
  {
    title: "Name",
    dataIndex: "name",
    hideInForm: true,
  },
  {
    title: "Age",
    dataIndex: "age",
  },
];

const data = [
  {
    key: "1",
    name: "John Brown",
    age: 32,
  },
  {
    key: "2",
    name: "Jim Green",
    age: 42,
  },
];

function App() {
  return <Table columns={columns} dataSource={data} />;
}

export default App;
```
