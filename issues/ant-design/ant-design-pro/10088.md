如果要在 `config/config.ts` 中修改侧边菜单栏的宽度，可以使用 `childrenLayout` 属性，示例如下：

```typescript
export default {
  layout: {
    childrenLayout: {
      siderWidth: 208, // 修改侧边菜单栏宽度
    },
  },
  // ...
};
```

此外，还可以将 `siderWidth` 属性写在 `defaultSettings.ts` 中或在 `app.tsx` 的 `layout` 属性中修改。
