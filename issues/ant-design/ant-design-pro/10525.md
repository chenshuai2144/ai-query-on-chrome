# 👑 [需求 | Feature]建议路由配置项中增加对多标签页面的配置

### 🥰 需求描述

config.ts中开启了多标签页，同时部分页面需要沉浸式导航，因此在route.ts中，对该页面做了对菜单导航的隐藏设置，但是发现目前的[配置项](https://umijs.org/docs/max/layout-menu)中缺少对多标签页的配置，导致沉浸式页面依然显示多标签栏

config.ts

```
  // 开启多标签页
  keepalive: [/./],
  tabsLayout: {
    hasDropdown: true,
  },
```

route.ts

```
  {
    path: '/test',
    name: '沉浸式页面',
    icon: 'smile',
    // 不展示顶栏
    headerRender: false,
    // 不展示页脚
    footerRender: false,
    // 不展示菜单
    menuRender: false,
    // 不展示菜单顶栏
    menuHeaderRender: false,
    component: './Test',
  },
```
