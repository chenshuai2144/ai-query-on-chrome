# 🐛 [BUG] 多标签显示功能显示异常

### 🐛 bug 描述

在启用多标签功能后，设置splitMenus:true，多标签显示异常，关闭也有问题

### 📷 复现步骤 | Recurrence steps

1、config配置：

```
  keepalive: [/./],
  tabsLayout: {},
```

2、defaultSettings配置

```
layout: 'mix',
splitMenus: true,
```

3、菜单结构：

```
  {
    name: 'test1',
    path: '/jlkd',
    icon: 'codeSandbox',
    routes: [
      { path: '/jlkd', redirect: '/jlkd/opr' },
      {
        name: 'test2',
        path: '/jlkd/opr',
        icon: 'codeSandbox',
        component: './JianLiangKongDa/OrderPurifyingRod',
      },
    ],
  },
```
