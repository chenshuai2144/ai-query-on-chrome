问题描述：在使用ProFormRadio.Group组件时，控制台提示"Incorrect use of <label for=FORM_ELEMENT>"。
示例代码：

```jsx
<ProFormRadio.Group
  name="sex"
  label="请选择性别"
  initialValue={"1"}
  options={[
    {
      label: "男",
      value: "1",
    },
    {
      label: "女",
      value: "2",
    },
  ]}
/>
```

其他信息：使用的是ProFormRadio.Group组件，版本号为3.0.10，操作系统为Windows 10，Node版本为14.15.3，浏览器为Chrome 89.0.4389.114。
