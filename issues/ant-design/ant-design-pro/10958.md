### 🧐 回答 | Answer

根据您的描述，问题可能是由于.vscode目录下的缓存文件引起的。您可以尝试删除.vscode目录下的缓存文件，并重新启动VSCode。

另外，您也可以尝试关闭VSCode的自动保存功能，以避免在项目启动时触发重新编译。您可以通过在VSCode的用户设置中添加以下配置来禁用自动保存：

```json
"files.autoSave": "off"
```

如果问题仍然存在，请确保您的项目依赖项和环境配置正确，并检查相关错误日志。您可以尝试使用以下命令重新安装依赖项：

```shell
npm install
```
