### Steps to reproduce

```javascript
const options = [
  { id: "id1", name: "name1" },
  { id: "id2", name: "name2" },
].map((data) => {
  return {
    key: data.id,
    label: data.name,
  };
});

<Breadcrumb.Item
  key="breadcrumb"
  menu={{
    items: options,
    onClick: (e) => {
      console.log(e.key);
    },
  }}
></Breadcrumb.Item>;
```

### What is expected?

console.log(key) key是 id1, id2

### What is actually happening?

console.log(key) key是 1, 2
