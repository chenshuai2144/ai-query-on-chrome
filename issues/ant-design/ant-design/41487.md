antd5中，DatePickerProps中有showTime属性
我自定义了个属性用于重写DatePickerProps的value属性
代码示例如下

```
import React from 'react';
import { DatePickerProps } from 'antd';

type PropsType = Omit<DatePickerProps, 'value'> & {
  value?: string;
};

const MyComp: React.FC<PropsType> => {...}

export default  MyComp;
```

```
//此时会报错，不存在showTime
< MyComp showTime/>
```
