# 问题描述

在使用antd的下拉选择组件时，如果在初始化时已经选择了某些数据，但在搜索或选择其他数据时，这些已选择的数据会被冲掉。

## 重现步骤

1. 输入关键字，搜索下拉选项。
2. 选中一条数据。
3. 再次输入关键字，搜索下拉选项。
4. 发现之前已选择的数据被冲掉。

## 期望结果

在搜索或选择其他数据时，已选择的数据不应该被冲掉。

## 实际结果

在搜索或选择其他数据时，已选择的数据被冲掉。

## 环境信息

- antd：undefined
- React：react 18
- System：mac
- Browser：chrome

      onChange={(newValue, arr) => {
--      setValue(arr);
