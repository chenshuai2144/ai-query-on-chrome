## MadCcc

`hashPriority="high"` 会提高 hash 的优先级，而不是降权
如果想要确保自己的魔改样式一定生效，可以在前面加上 id 选择器来保证优先级

```css
#app .custom-class {
}
```

可以通过 postCSS 来统一处理
