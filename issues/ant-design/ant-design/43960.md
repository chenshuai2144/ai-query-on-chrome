# FormItemProps children 为函数时，FormInstance类型来自rc-field-form,期望FormInstance类型从antd引入

`🤔 Need Reproduce`,`unconfirmed`

### Reproduction link

### Steps to reproduce

import { Form, Input, type FormInstance } from 'antd';

<Form.Item>
(form: FormInstance) => {
return (
<Form.Item name="1">
<Input />
</Form.Item>
);
}

</Form.Item>

### What is expected?

FormItemProps['children']的类型为React.ReactNode | (form: FormInstance<Values>) => React.ReactNode;
此处的FormInstance类型与实际类型不一致，此处FormInstance引入自rc-field-form

### What is actually happening?
