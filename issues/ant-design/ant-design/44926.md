### Steps to reproduce

```html
<Form.Item labelCol={{ span: 5 }} wrapperCol={{ span: 22 } } name="sourceLimit" label="请求次数数量上限">
      <>
        <InputNumber placeholder='请输入请求数量上限' style={{ width: '50%' }} />
        <span>tian</span>
     </>
</Form.Item>
```

这段代码里sourceLimit的值将无法被读取，去除掉<span>tian</span>就可以了

### What is expected?

可以正确读取到对应的值

### What is actually happening?

无法读取值
