## MadCcc

由于 antd 的样式也自带具有优先级的选择器，目前只能够手动提高自定义样式的选择器优先级，tailwind 可以选择开启 important。

你也可以使用这种方法来全局提升你的 css 优先级：

```diff
--  .my-btn {
++  #root .my-btn {
      background: red;
    }
```

Ref: https://ant-design.antgroup.com/docs/react/compatible-style-cn#where-%E9%80%89%E6%8B%A9%E5%99%A8
