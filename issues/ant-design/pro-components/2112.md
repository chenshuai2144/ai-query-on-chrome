# 👑 [需求] ProColumns 的 valueType: 'select' 能否支持参数showSearch

### 🧐 问题描述

ProColumns 的 valueType: 'select' 能否支持参数showSearch，
select 标签有参数showSearch 可以支持搜索，但ProTable的ProColumns 好像不支持这个参数，能否增加下支持。

```
      },
},
```
