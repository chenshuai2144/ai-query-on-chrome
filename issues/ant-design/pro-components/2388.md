## wuhao930406

> 抱歉我理解错了，请问有考虑支持移动端点击后自动折叠菜单么
> 可以在 app.ts

```javascript
export async function getInitialState() {
  return {
    name: "@umijs/max",
    collapsed: false,
  };
}

export const layout = ({ initialState, setInitialState }) => {
  return {
    logo: Settings?.logo,
    collapsed: initialState?.collapsed,
    onCollapse: (collapsed) => {
      setInitialState((s) => ({
        ...s,
        collapsed,
      }));
    },
    onPageChange: () => {
      setInitialState((s) => ({
        ...s,
        collapsed: true,
      }));
    },
    menu: {
      locale: false,
    },
  };
};
```

onPageChange时强制收起菜单
