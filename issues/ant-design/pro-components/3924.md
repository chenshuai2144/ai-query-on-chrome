## qqjay2017

这么写成功了

```tsx
submitter={{
            searchConfig:{
              submitText:intl.formatMessage({ id: 'login.LogIn' }),

            },
            submitButtonProps:{
              style:{
                width:'100%'
              }
            },
            resetButtonProps:false
          }}
```
