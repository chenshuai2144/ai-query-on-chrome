## Ortonzhang

这样写你的ProFormSelect。添加noStyle 属性

```jsx
<ProFormDigit
  name={["userLimit", "type"]}
  label="用户限购"
  addonAfter={
    <ProFormSelect
      name={["userLimit", "number"]}
      noStyle
      request={async () => [
        { value: "forever", label: "永久" },
        { value: "day", label: "每日" },
        { value: "week", label: "每周" },
        { value: "month", label: "每月" },
      ]}
    />
  }
/>
```
