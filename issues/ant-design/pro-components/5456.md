BetaSchemaForm layoutType 为 StepsForm 时 layout 参数设置为 horizontal 不生效
源码里面写死了 vertical ,而按照客户的需求，需要表单标签水平排列

            layout: 'horizontal',
            labelCol: { span: 6 },
            wrapperCol: { span: 18 },

        ]}`

## Laurieliuren
