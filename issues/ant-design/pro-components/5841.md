# 🐛[BUG] layout 内容区域超出屏幕区域

## 🐛 bug 描述

布局内容区域超出屏幕区域。

## 📷 复现步骤

1. 使用 "@ant-design/pro-components": "^2.3.2" 版本的组件。
2. 将布局内容区域的 margin-inline 设置为 inherit，使其超出屏幕区域。

## 🏞 期望结果

布局内容区域应保持在屏幕区域内。

## 💻 复现代码

```css
.ant-pro-layout-content {
  margin-inline: inherit !important;
}
```
