# 🧐[问题]ProTable自定义搜索文本框会被赋上max-width: cale(100%-search.labelWidth)内联样式

`ProTable` 自定义字段搜索文本框会被赋上`max-width: cale(100%-search.labelWidth)`内联样式

```tsx
renderFormItem: () => {
  return (
    <ProFormFieldSet
      name="list"
      type="group"
    >
      <Input style={{ width: "50%" }} />
      <Input
        style={{
          width: "50%",
        }}
      />
    </ProFormFieldSet>
  );
},
```

```tsx
search={{
  labelWidth: 124,
}}
```

## ByYogi

暂时解决方案

```tsx
.ant-form-item-control {
  max-width: none !important;
}
```
