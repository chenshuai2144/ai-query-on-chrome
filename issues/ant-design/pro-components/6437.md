如果您使用的是Ant Design的Form组件，可以尝试以下做法：

1. 设置搜索表单单独占一行

您可以使用`Form.Item`组件的`wrapperCol`属性将搜索表单布局到单独的一行。

```jsx
<Form layout="inline">
  <Row>
    <Col span={24}>
      <Form.Item label="关键词" name="search">
        <Input />
      </Form.Item>
    </Col>
  </Row>
  <Row gutter={24}>
    <Col>
      <Form.Item>
        <Button type="primary" htmlType="submit">
          查询
        </Button>
      </Form.Item>
    </Col>
    <Col>
      <Form.Item>
        <Button>导出</Button>
      </Form.Item>
    </Col>
    <Col>
      <Form.Item>
        <Button>重置</Button>
      </Form.Item>
    </Col>
  </Row>
</Form>
```

2. 将搜索表单展示为下拉框

对于过多的操作按钮，您可以考虑将其展示为下拉框，这样可以更好的节省空间。

```jsx
<Form layout="inline">
  <Form.Item label="关键词" name="search">
    <Input />
  </Form.Item>
  <Form.Item style={{ marginLeft: "auto" }}>
    <Dropdown overlay={menu}>
      <Button>
        更多操作 <DownOutlined />
      </Button>
    </Dropdown>
  </Form.Item>
</Form>
```

更多关于Ant Design Form的使用和API，请查看官方文档：https://ant.design/components/form-cn/ 。
