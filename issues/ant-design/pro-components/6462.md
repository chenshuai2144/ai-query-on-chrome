```js
<PageContainer
  breadcrumb={{
    routes: [
      {
        path: "/customers",
        breadcrumbName: "客户管理",
      },
      {
        path: "/customers/:customerId",
        breadcrumbName: "客户详情",
      },
    ],
    itemRender: (routes) => (
      <Link to={routes.path}>{routes.breadcrumbName}</Link>
    ),
  }}
/>
```

如上方式设置面包屑，在本地开启 mfsu 后样式正常，本地关闭 mfsu 或构建后的样式丢失（.antd-breadcrumb 都没有打包进来），目前手动在项目中引入兼容性代码可以解决：`import 'antd/es/breadcrumb/style';`
