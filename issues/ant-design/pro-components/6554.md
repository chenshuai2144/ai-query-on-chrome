# 🐛bug

在protable中使用`ellipsis`属性和`render`的第一个参数时，代码执行错误。

### 📷复现步骤

随便创建一个protable就可以。

### 🏞期望结果

text为任何值都只会显示 '下游'，删除 ellipsis: true,则显示正常。

### 💻复现代码

```javascript
{ title: "类型", dataIndex: "type", search: false,
    ellipsis: true,
     render: (text:any) => text == 'producer'?<div>上游</div>:<div>下游</div>
}
```
