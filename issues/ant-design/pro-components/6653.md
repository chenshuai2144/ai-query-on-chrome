您好，根据您提供的信息，似乎是Antd升级到5版本后，表单中的图标大小发生了变化。这可能是由于Antd 5版本对CSS样式层级优化所致导致的，因此您可以手动指定样式层级以覆盖默认的样式。

以下是一个可能的解决方案，您可以在样式表中添加以下CSS规则：

```css
.ant-form-item-label > .anticon:before {
  font-size: 16px !important; /* 指定图标大小 */
  margin-right: 8px; /* 指定图标右边距离表单项的距离 */
}
```

其中，`.ant-form-item-label` 和 `.anticon:before` 分别代表 Antd 表单项的标签和图标区域。

此外，您可能还需要考虑指定其他相关的样式，以更好地符合您的设计需求。

另外，您可以参考 Antd 官方文档：https://ant.design/components/form-cn/#components-form-demo-login ，以获取有关如何使用 Antd 表单组件的更多信息。
