你好，根据你提供的信息，可能是由于添加了 HeaderTitle 后导致了 ProList 的高度变化，从而出现了滚动条和样式问题。

解决方案：

可以通过设置 ProList 的高度为自适应，使得它的高度会根据内容自动调整，从而避免这个问题。具体做法如下：

```jsx
import { ProList } from "@ant-design/pro-list";

<ProList
  headerTitle="ProList"
  // 设置高度为 auto
  style={{ height: "auto" }}
>
  {/* content */}
</ProList>;
```
