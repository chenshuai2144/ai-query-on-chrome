根据官网提示，Chrome 88 版本之前的浏览器将不再兼容 React。但对于已经安装的老版本的浏览器，您仍可以使用 polyfill 等工具来提供相应的支持。

具体来说，您可以使用 babel 和 @babel/preset-env 插件来转译代码，在打包构建时使用 core-js 和 regenerator-runtime 等 polyfill 转换器进行代码适配，从而兼容较老浏览器。

以下是一个简单的示例，仅供参考：

```js
// .babelrc

{
  "presets": [
    [
      "@babel/preset-env",
      {
        "targets": {
          "browsers": [
            "last 2 versions",
            "not ie <= 10"
          ]
        }
      }
    ],
    "@babel/preset-react"
  ]
}

// package.json
{
  "dependencies": {
    "@babel/core": "^7.12.10",
    "@babel/preset-env": "^7.12.11",
    "@babel/preset-react": "^7.12.10",
    "core-js": "^3.10.1",
    "regenerator-runtime": "^0.13.7"
  }
}

// webpack.config.js
module.exports = {
  // ...
  module: {
    rules: [
      {
        test: /\.js$/,
        exclude: /node_modules/,
        use: {
          loader: "babel-loader",
          options: {
            cacheDirectory: true
          }
        }
      }
    ]
  }
};
```
