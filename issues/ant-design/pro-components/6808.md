### 🐛 bug 描述

ProDescriptions 使用columns属性的时候，如果有的columns是valueType是digit、money等，垂直方向不能居中显示，跟align-items是baseline有关。

### 📷 复现步骤

<ProDescriptions
dataSource={{
                    rate: 5,
                    id: '这是一段文本columns',
                    date: '20200809',
                    money: '1212100',
                    state: 'all',
                    state2: 'open',
                }}
editable={{}}
columns={[
{
title: '分值',
dataIndex: 'fraction',
valueType: "digit"
}
]}
/>

点击编辑按钮，然后看打钩和打叉两个按钮的垂直位置
