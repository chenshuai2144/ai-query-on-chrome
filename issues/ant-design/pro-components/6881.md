可以用第三个参数操作一下

```
 menuItemRender={(item, dom, props) => (
        <a
          onClick={() => {
            props.onCollapse?.(true);
            setPathname(item.path || '/welcome');
          }}
        >
          {dom}
        </a>
      )}
```
