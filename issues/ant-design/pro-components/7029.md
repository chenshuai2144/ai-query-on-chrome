```
syncToUrl: (values, type) => {
         if (type === "get") {
           console.log(values, "=============");
           return {
             ...values,
             betNum: [values.betBegin, values.betEnd]
           };
         }
         return values;
       },
```

这个 get 和 set 都要设置的，因为url里面没有具体的数据结构，你要转化下
