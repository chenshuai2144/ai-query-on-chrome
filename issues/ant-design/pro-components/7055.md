# ProTable使用编辑功能，修改tableListDataSource，UI没有对应修改🐛[BUG]

### 🐛 bug 描述

ProTable使用编辑功能，当外部动态修改了tableListDataSource，ProTable的UI没有对应的修改。如果将startEditable一行注释掉，那么ProTable的UI可以正常变换
