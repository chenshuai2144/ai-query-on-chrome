# ProFormText 设置hidden属性导致name无效

## 🐛 Bug 描述

设置ProFormText的hidden属性后，name属性无法正常渲染。

## 📷 复现步骤

1. 在ProFormText中设置hidden属性。
2. 观察name属性是否正常渲染。

## 🏞 期望结果

name属性需要正常渲染。

## 💻 复现代码

```jsx
import { ProFormText } from "@ant-design/pro-form";

function App() {
  return <ProFormText hidden name="test" />;
}
```
