# 🐛[BUG] dateFormatter="number" 没有正确的序列化时间

### 🐛 bug 描述

EditableProTable和ProForm 设置dateFormatter="number"时表单没有将时间序列化时间戳.

### 📷 复现步骤

使用官网EditableProTable演示代码,设置dateFormatter="number"

### 🏞 期望结果

时间类型被正确的序列化为时间戳.

### 💻 复现代码

见官网EditableProTable演示代码

- antd 版本: [5.6.1]
- 浏览器环境
