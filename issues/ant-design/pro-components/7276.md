# 👑 [需求] 希望ProCard可以通过theme components 来修改borderRadius大小

提问前先看看：

https://github.com/ryanhanwu/How-To-Ask-Questions-The-Smart-Way/blob/main/README-zh_CN.md

### 🔩 所属模块或组件

ProCard

### 🥰 需求描述

目前ProCard的圆角只能通过token.borderRadius设置。

```js
// 希望可以通过 components 来修改ProCard圆角大小。
theme: {
    components: {
      Card: {
        borderRadiusLG: 20,
      },
    },
  },
```

### ⛰ 功能需求适用场景

只需要修改ProCard的圆角大小。

### 🧐 解决方案

是否可以移植[antd](https://github.com/ant-design/ant-design/blob/master/components/theme/util/genComponentStyleHook.ts)的方法只修改card的样式，不是修改所有组件的样式？
