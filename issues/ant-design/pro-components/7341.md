# ProTable时DataSource的数据格式不支持string[]

### 🐛 bug 描述

当使用ProTable时DataSource的数据格式为string[]等基础数据类型时会报错，此bug是因为使用了Reflect.has方法导致，此方法不支持基本类型

### 🏞 期望结果

可以能支持基础数据类型
