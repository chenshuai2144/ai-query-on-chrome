# 🧐[问题]如何获取路由上的参数

### 🧐 问题描述

有时候需要像react-router一样定义一个路由上的参数，比如说/:id/xxx，想要获取这个id参数

```js
<Route path="/edit/:id" component={Editor} />
```

```js
//Editor.jsx
function Editor({ match }) {
  console.log(match.params.id);
}
```
