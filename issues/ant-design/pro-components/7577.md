# 👑 [需求] ProTable 搜索表单条件改变时自动触发查询

### 🔩 所属模块或组件

ProTable

### 🥰 需求描述

在后台系统中，经常会有切换状态值、时间范围、下拉选择框等搜索条件，希望能修改值后自动触发查询。

目前也可以通过设置 onValueChange 再调用 formRef.submit 去更新，但这样每个表格都需要设置，挺不优雅的。

### 🧐 解决方案

建议在 ProColumns 中加个设置，是否允许修改值后触发查询，或自动触发查询的类型（比如：change、blur）
