# 🐛[ProTable] 搜索表单， collapse不需要时，样式存在问题。

### 🐛 bug 描述

当不需要渲染展开收起时，没有提供showCollapse这样的api控制，是通过collapseRender来处理，此处包裹组件通过space+内联style:{gutter:16}处理的，此处处理不当，有样式bug
