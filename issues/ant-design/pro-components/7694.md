# 🐛[BUG] `ProFormUploadButton` 组件 `onChange` 事件上传触发两次

## 📷 复现步骤

1. 使用 `ProFormUploadButton` 组件
2. 触发 `onChange` 事件

## 🏞 期望结果

`onChange` 事件只触发一次

## 💻 复现代码

```jsx
<ProFormUploadButton
  label="选择文件"
  title="导入"
  accept=".xlsx"
  fieldProps={{ showUploadList: false, headers: { Authorization: getLocalStorage('TOKEN')! } }}
  onChange={(info)=>{
    console.log(info);
  }}
  action={`/api/teacher`}
  icon={<ImportOutlined />}
/>
```
