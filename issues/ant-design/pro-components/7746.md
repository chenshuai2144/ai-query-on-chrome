## chenshuai2144

protable 的 search 是个 form ，是可以用 formRef 来完成你想要的操作。

1： formRef.current?.getFieldFormatValue();
2： formRef.current?.submit();

```

export default () => {
  const formRef = useRef<ProFormInstance>();

  return (
    <ProTable<TableListItem>
      formRef={formRef}
      dateFormatter="string"
      headerTitle="表单赋值"
    />
  );
};
```
