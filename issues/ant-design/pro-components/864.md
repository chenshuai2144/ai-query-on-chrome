```bash

yarn

yarn dev

http://0.0.0.0:9090/#/pro-table/table-2
```

`reset` 重置 Table columns 中所有的 filteredValue。

`random data` 模拟请求数据，更新列表。

ProTable 的表现是没有把 `type` 的筛选条件清空(UI 上的样子并没有恢复未选择的样子)。
