# [Bug] mfsu 拦截 .json 并有缓存的问题

如何复现？

1、git clone git@github.com:sorrycc/weekly.git
2、安装依赖
3、把 mfsu 开回来，npx umi config set mfsu {}
4、npx umi dev
5、访问 /posts.json 。
6、修改 public/posts.json 文件
7、刷新 /post.json，会发现改动没有生效，期望是改动生效

而如果把 mfsu 关闭，重复 5-7 是会生效的。
