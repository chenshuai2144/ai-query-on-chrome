# Bug: SelectLang render 为空

## What happens?

在配置locale后，执行`yarn dev`，生成的plugin-locale代码中SelectLang组件渲染为空。

## How To Reproduce

配置locale，然后执行`yarn dev`，查看生成的 plugin-locale 代码。

## Context

- Umi Version: Umi v4.0.15
- Node Version: 16.18.0
- Platform: Windows10

## Solution

检查`locales/*`下的语言文件完整性和语法规范性。
