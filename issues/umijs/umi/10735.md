## fz6m

## fz6m

See https://umijs.org/docs/guides/use-plugins

```ts
// .umirc.ts

export default {
  plugins: ["@umijs/plugins/dist/locale"],
  locale: {},
};
```

umi 配置插件就是 max 一样的功能，max 是由一个个插件组成的。
