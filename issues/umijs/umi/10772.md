## What happens?

开发 插件时，发现api.onStart 会多次执行，期望只执行一次

## Mini Showcase Repository(REQUIRED)

```ts
// plugin.ts
import { IApi } from "umi";

export default (api: IApi) => {
  api.onStart(() => {
    console.log("Local Plugin");
  });
};
```
