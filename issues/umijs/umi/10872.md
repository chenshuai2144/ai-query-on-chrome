## zenotsai

现在默认是 `root`，没读取 `.umirc` 中的配置

你可以用[页面级配置](https://umijs.org/docs/guides/mpa#configjson)来实现想要的效果

```js
// src/pages/Index/config.json
{
  "mountElementId": "test"
}

```
