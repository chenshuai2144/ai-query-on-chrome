配置如下：

```
cacheGroups:{
    vendors: {
            chunks: 'all',
            priority: -10,
            enforce: true,
            test: /[\\/]node_modules[\\/]/,
            name({ context: modulePath }) {
              const handleModuleContext = modulePath?.match(MODULE_PATH_REGEX);
              if (!handleModuleContext) {
                return false;
              }
              const [, , scope, name] = handleModuleContext;
              const packageName = ['npm', (scope ?? '').replace('@', ''), name]
                .filter(Boolean)
                .join('.');
              return packageName;
            }
    }
}
```
