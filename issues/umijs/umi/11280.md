# .umi 文件夹下 tsconfig.json 文件 paths 应该包含 umi 内置及配置的 alias

## Background

当我使用 import { useSelector } from '@umijs/max' 时，useSelector 类型错误问题

## Proposal

内置和配置项的 alias 应该在 tsconfig.json 模版中 的paths 字段体现

## fz6m

你看到的 `dva` 、`antd` 等内部插件添加的 alias 是为了定位依赖唯一性使用的，所有的 dva 方法都在 umi 导出了，所以你只需要从 umi 导入使用即可。
