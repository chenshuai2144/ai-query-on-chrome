# Bug: UmiJs4 Sse无法实时显示

## What happens?

在使用代理模式下，sse不能实时返回数据，而是需要等sse服务端关闭才会返回数据。

## How To Reproduce

1. 使用代理模式运行项目
2. 发送sse请求
3. 等待sse服务端关闭，然后才能收到数据

## Expected behavior

sse应该能够实时返回数据，无需等待服务端关闭。

## Context

- Umi Version: 4.x
- Node Version: 14.x
- Platform: Any
