## fz6m

如果你用了 umi layout 插件（max 项目），这个 layout 就需要依赖 pro component ，不用在意它背后是如何依赖的。

如果你发生了 react 多实例问题，可以尝试升级 umi 至最新版本，排查下是不是依赖错乱导致的，删除 `node_modules` 重装依赖，再尝试看看行不行。

如果还不行，可以尝试关闭 `mfsu: false` 。

最后还有问题的话，需要提供一个最小复现帮助排查问题。
