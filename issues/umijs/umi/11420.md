# [Bug] umi添加插件报错

## 问题描述

在umi 4中，手动添加插件时出现错误，提示`Cannot read properties of undefined (reading 'winPath')`。

## 重现步骤

1. 安装插件：`npm i -D @umijs/plugins/dist/model`
2. 在`.umirc.ts`中添加插件：`export default { plugins: ["@umijs/plugins/dist/model"], model: {}, };`
3. 运行`umi dev`命令

## 期望行为

插件能够正常添加并运行。

## 相关配置

- Umi版本：latest
- Node版本：16
- 平台：无

## 解决方案

将插件改为`@umijs/plugins/dist/model`。
