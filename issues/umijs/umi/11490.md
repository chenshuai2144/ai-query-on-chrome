## How To Reproduce

1. 安装node16以后的模块： 例如

```
yarn add @aomi/react-pro-ant-design
```

2. 在page中引入模块

```
import {QueryContainer} from "@aomi/react-pro-ant-design/QueryContainer";

export default function Dashboard() {

  const columns = [{
    title: "Title"
  }]

  return <QueryContainer table={{
    columns
  }}/>
}

```
