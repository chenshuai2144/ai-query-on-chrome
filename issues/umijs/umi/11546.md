# [Bug] 设置了request 的 timeout，请求无法超时cancel

`type(umi@3)`

## What happens?

```js
import { request } from 'umi';

export async function someRequest(data) {
  return request(`xxx`, {
    method: 'POST',
    data,
    timeout: 2000 //  <-   这里设置超时时间
  });
}

客户端浏览器无法在在指定的timeout时cancel请求，请求会持续走完直到服务端响应

umi版本  "umi": "^3.5.0"

```
