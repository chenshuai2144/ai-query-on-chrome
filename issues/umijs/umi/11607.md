# Bug: Command failed with ENOENT: umi g

## How To Reproduce

1. Run `pnpm dlx create-umi@latest`
2. Choose to use Ant Design Pro
3. Run `pnpm umi g`

## Context

- Umi Version: 4.0.79
- Node Version: 18.16.0
- Platform: Mac M2
- pnpm Version: 8.3.1

## Solution

Use `pnpm max g` instead of `pnpm umi g`.
