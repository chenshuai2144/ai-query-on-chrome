# 如何在antd运行时配置函数中，异步的调用接口，获取主题数据。目前看antd运行时配置函数中不支持异步，请问我想异步获取异步数据应该如何做？

如何在antd运行时配置函数中，异步的调用接口，获取主题数据。目前看antd运行时配置函数中不支持异步，请问我想异步获取异步数据应该如何做？

## fz6m

对于运行时配置要异步获取配置的场景，目前只能在 render 前调用接口获取到数据后再执行 render ，这样后续就可以获取到这个数据了：

```ts
// src/app.ts

let globalConfig = null;

export const render = (oldRender: any) => {
  api().then((res) => {
    globalConfig = res;
    oldRender();
  });
};

export const antd = () => {
  // use globalConfig
};
```
