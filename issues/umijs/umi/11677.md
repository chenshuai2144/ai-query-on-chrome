# [Bug] 使用cloudflare 域名后，刷新页面微前端的子应用抱错 502

`type(duplicate)`

有两个项目 frame 和 oms，通过 qiankun 微前端来关联。
frame config 配置是：
{
path: '/',
redirect:'/oms'
},
{
path:'/oms/\*',
microApp:'oms',
microAppProps: {
autoSetLoading: true,
},
}
这样访问frame url "xxx.com" 自动跳转到 oms 系统 "xxx.com/oms" 是work 的。
但是我直接刷新 “xxx.com/oms” cloudflare 返回502 的错误。
我把 cloudflare 换成 godaddy 就没问题了。cloudflare 里面的 内容压缩 和加速我也都关闭了。
