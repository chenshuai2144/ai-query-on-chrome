## fz6m

你需要 viewBox 做啥？这是 svgo 的默认 svg 优化策略。

```ts
// .umirc.ts

export default {
  svgo: {
    plugins: [
      {
        name: "preset-default",
        params: {
          overrides: {
            // more options see https://github.com/svg/svgo#built-in-plugins
            removeViewBox: false,
          },
        },
      },
      "prefixIds",
    ],
  },
};
```
