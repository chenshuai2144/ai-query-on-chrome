## Context

- **Node Version**: v18.7.0
- **Platform**: chrome

  ## fz6m

  umi 里 use model 的东西现在都是全局独立存在的，所以它才可以保持持久性在各个组件共享，model 里拿不到应用里的上下文。

此处可以下放逻辑到局部的 context 去，或者使用 [hox](https://github.com/umijs/hox) ，该方案可以让逻辑局部持久存在，而不是像 umi 的 model 一样存在于全局拿不到上下文。
