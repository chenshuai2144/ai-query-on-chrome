## fz6m

umi 4 使用该插件的方法：

1. 确保安装了 `@alita/plugins` 这个依赖。

2. 在配置文件中打开该插件：

   ```ts
   // .umirc.ts

   export default {
     plugins: ["@alita/plugins/dist/hd"],
     hd: {},
   };
   ```
