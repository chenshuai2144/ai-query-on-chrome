## fz6m

最新版本已经解决了这个问题：

```diff
// package.json

  "dependencies": {
+   "@ant-design/pro-components": "^2.3.30",
+   "@umijs/max": "^4.0.30"
  }
```

```diff
// .umirc.ts

    {
      name: ' 一级路由',
      path: '/route',
-     component: './Table',
      routes: [
        {
          name: ' 二级路由1',
          path: '/route/test1',
          component: './Table',
          // 去掉这一行菜单就会正常展示
          wrappers: ['@/wrappers/Wrapper.tsx'],
        },
        {
          name: ' 二级路由2',
          path: '/route/test2',
          component: './Table',
        },
      ],
    }
```
