# [Feature Request] @umijs/ast 中 setConfigByName 无法设置嵌套对象

## Background

实现自定义生成器时，使用内部方法无法设置嵌套对象

## 实现

希望可以支持复杂对象

```ts
setConfigByName(
  ast,
  "plugins",
  JSON.stringify({
    remotes: [
      {
        name: "foo",
        aliasName: "bar",
        keyResolver: '"key1"',
        entries: {
          key1: "http://a.b/c.js",
          key2: "http://a.b/c.js",
        },
      },
    ],
  }),
);
```
